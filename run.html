Run

Info

Run is an endless runner game.
Created by Summer Rain for CISGAME2 using the Construct 3 engine.


Rules & Controls
Use your finger or mouse to navigate through the game.
Tap(click) to jump.


*NOTES*
This game wen through development hell.
I started this game off as a standard platformer, but no matter what I did it just wasn't fun.
After some playtesting I found out that people had a little bit of fun trying to beat the clock, so I turned it into a speedrunning challenge and shifted the focus to beating the clock.
But even this change wasn't very fun.
Eventually, I decided to scrap the platformer soncept and work on an endless runner with the same assets.
I think this version is much more accessible and entertaining.
